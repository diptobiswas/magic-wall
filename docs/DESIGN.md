# Design

Magic Wall should feel like an ambient object first and an annotated artwork second.

## Principles

- Keep the generated image as the default full-screen view.
- Reveal information only after touch, then make it easy to hide again.
- Use dense information only where it explains the current wallpaper.
- Favor big touch targets over tiny controls.
- Keep text short and useful.
- Use motion for state changes, not decoration.

## Story Overlay Layout

- Hidden by default over the artwork.
- Header: status, time, provider, hide.
- Left: selected story title, summary, significance, source link.
- Right: artwork metadata such as source, published time, generated time, next refresh, cadence, and style.
- Footer: story title, generated time, manual art refresh.

## Visual Tone

Dark glass panels over bright artwork, one sharp lime accent, and restrained metadata rows.
