# Design

Magic Wall should feel like an ambient object first and a dashboard second.

## Principles

- Keep the generated image as the default full-screen view.
- Reveal information only after touch, then make it easy to hide again.
- Use dense information only where it helps quick scanning.
- Favor big touch targets over tiny controls.
- Keep text short and useful.
- Use motion for state changes, not decoration.

## Dashboard Layout

- Hidden by default over the artwork.
- Header: status, time, provider, hide.
- Left: strongest selected signal.
- Middle: category tabs and touch cards.
- Right: X pulse links.
- Footer: art metadata, check, art refresh, X sheet.

## Visual Tone

Dark glass panels over bright artwork, one sharp lime accent, and small secondary heat colors for signal state.
